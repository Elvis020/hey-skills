# Example Prompts

Use these prompts when you want the skill to frame a real decision:

- Should we build auth or use Clerk?
- Is this a monolith problem or a microservices problem?
- Should we rewrite this service or refactor it incrementally?
- Is TypeScript worth it for this codebase right now?
- Should we optimize this query or ship and measure first?
- What should we cut from v1 to launch in two weeks?
